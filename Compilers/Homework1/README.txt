Compilers

Part1:
Στο part1 έχω φτίαξει τον lexer ο οποίος διαβάζει τα σύμβολα και τα στέλνει στον parser.
Ο lexer είναι βασισμένος στα παραδείγματα που έχει ο cup που μας δίνεται link στο μάθημα.
Το πρόγραμμα το έχω κάνει να δέχεται ένα input.txt όπου η κάθε πράξη ξεχωρίζει με αλλαγή γραμμής εκτος από
την τελευταία.

δλδ java Parser < input.txt (προφανώς ότι όνομα θέλουμε έχει το .txt) 

Όπότε ο parser σε μια λουπά διαβάζει τα σύμβολα της κάθε πράξης και ανά σύμβολο καλεί της συναρτήσεις
της γραμματικής που έχουμε και τοποθετεί τους αριθμούς κατάλληλα στο stack μας και ανάλογα τα σύμβολα
γίνονται και τα κατάλληλα pop() βάση της γραμματικής μας.
Επίσης έφτιαξα την κλάση symbol για να αποθηκεύω αυτήν στο stack. 

Part2:
Στο παρτ2 έχω φτιάξει έναν lexer/scanner όπου είναι .flex και επιστρέφη τα σύμβολα στον parser.cup.
Όπως και στο παρτ1 έχω κάνει το πρόγραμμα να δέχεται ενα input.txt και να το κάνει print. Αν θέλουμε 
να πέρασουμε τον παραγόμενο κώδικα σε ενα αρχείο .java τότε εκτελούμε: (οπώς τα παραδείγματα που δώθηκαν)
java -cp ../java-cup-11b-runtime.jar:. Main < input.txt > output.java
με το όνομα output δεν μπορεί να εκτελεστεί το αρχείο αλλά στον φάκελο μας υπάρχει ήδη μια Main

Στον parser έχω φτίαξει την γραμματική για να μπορώ να παίρνω ως input τα παραδείγματα και με κώδικα java
τα φέρνω στην μορφή και γραμματική που πρέπει να είναι για να είναι εκτελέσημα.

το methods είναι για της συναρτήσεις του κώδικα.
το expr_list είναι για να διαχειρίζεται το expr.
και στο expr υπάρχουν τα πάντα από ένα απλό + μέχρι και εμφολευμένα if/else

Προσπάθησα να πετύχω και την τέλεια στοίχιση για να υπάρχει ομορφία στο output και το κατάφερα 
για τα examples #1 και #2 στο example #3 για καποίο λόγο τα if δεν έχουν τέλεια στοίχιση.

Και ο scanner και ο parser ειναι βασισμένοι στα παραδείγματα που δώθηκαν στο μάθημα.

Είναι η πρώτη φόρα που παίρνω το μάθημα και ασχολήθηκα με compilers άρχισα της εργασίες σχετικά
αργά όποτε θέλω να σας ευχαριστήσω που δόθηκε η παράταση για την πρώτη εργασία ώστε φοιτητές σαν
εμένα να μπορούν να συμμετάσχουν και να ειναι έτοιμοι για τις επόμενες εργασίες.
